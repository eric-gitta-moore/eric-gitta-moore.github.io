# 5. 安装cli client

> python-designateclient包在 PyPI 上发布，因此可以使用 pip 工具安装，该工具将管理所有 python 依赖项的安装： 

可以发现没有 openstack zone create命令的原因是没有安装这个软件包

```
pip install python-designateclient
```
