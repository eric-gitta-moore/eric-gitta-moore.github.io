Manila 是 OpenStack 共享文件系统服务，用于将共享文件系统作为服务提供。马尼拉的一些目标是/拥有：

  • 基于组件的架构：快速添加新行为
  • 高可用：扩展到非常严重的工作负载
  • 容错：隔离进程避免级联故障
  • 可恢复：故障应该易于诊断、调试和纠正
  • 开放标准：成为社区驱动的 api 的参考实现
